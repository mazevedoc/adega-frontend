## 📌 Descrição

Este é um sistema de ERP voltado para gestão de adegas, com o objetivo de facilitar o controle de vendas, usuários e produtos de forma eficiente e com visual moderno.

🎯 O projeto visa aplicar boas práticas de desenvolvimento web full stack, com autenticação de usuários, painel administrativo e integração com banco de dados.

---

## 🧾 Funcionalidades

- Tela de login com validação de e-mail e senha
- Cadastro de usuários com verificação de CPF e e-mail
- Painel de administração para controle de:
  - 📦 Produtos
  - 👥 Usuários
  - 🧾 Mensagens recebidas via formulário
- Layout responsivo com foco em experiência do usuário
- Back-end modularizado com Node.js + Express
- Integração com PostgreSQL e autenticação via JWT

---

## 🛠 Tecnologias Utilizadas

### 💻 Front-end
- **HTML5** – Estrutura das páginas
- **CSS3 + Bootstrap** – Estilização moderna e responsiva
- **JavaScript (ES6+)** – Lógica da interface e integração com API

### 🖥️ Back-end
- **Node.js** – Ambiente de execução
- **Express** – Framework de rotas e middlewares
- **PostgreSQL** – Banco de dados relacional
- **JWT** – Autenticação via tokens
- **Render** – Hospedagem do back-end e banco
